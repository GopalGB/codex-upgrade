# prod-eisenhower-matrix

Plot every task on two axes-Important (advances your goals/values) vs Urgent (demands attention now). Four quadrants drive four actions: **Q1 Urgent+Important -> Do** (crises, hard deadlines). **Q2 Not-urgent+Important -> Schedule** (planning, deep work, relationships, prevention-where real leverage lives). **Q3 Urgent+Not-important -> Delegate** (most interruptions, many meetings, others' priorities masquerading as yours). **Q4 Neither -> Delete** (busywork, doomscrolling). The technique that makes this work: actively defend Q2 by time-blocking it FIRST each week before Q1/Q3 floods in-Covey's 'big rocks' principle. The diagnostic failure mode is the **urgency trap**: treating all urgent things as important. Most Q3 items feel important only because they're loud. Ask 'if I don't do this, what actually breaks?' If nothing breaks, it's Q3/Q4. Run this as a 5-minute morning triage, not a one-time exercise; urgency status changes hourly while importance rarely does.

**Tools:** Eisenhower matrix, urgent/important quadrants, Q2 scheduling
