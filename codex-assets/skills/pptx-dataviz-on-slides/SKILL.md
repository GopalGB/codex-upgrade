# pptx-dataviz-on-slides

Exec/consulting charts tell one story per chart with the conclusion labeled on it. Core forms: Waterfall/Bridge (Insert > Chart > Waterfall, or think-cell) to decompose a change from start to end — bars for each driver, set Total markers; CAGR/growth arrows annotating start-to-end rate; stacked columns WITH segment values AND a total above each column (native stacked hides totals — add a total data label trick or use think-cell which does it natively); Mekko/marimekko for share-of-share (width = segment size, height = %). If your org has think-cell, use it — it auto-draws waterfalls, Mekkos, CAGR arrows, and links to Excel with one-click refresh; otherwise approximate with native Combo charts + manual callout shapes. Decluttering rules: kill gridlines/borders/3D, label points directly (drop the axis), highlight the one bar that matters in an accent color and gray the rest, sort by value, round numbers to 2 significant figures. Add a single text 'so-what' callout per chart. Pitfall: dual axes that imply false correlation, and pies with >4 slices (use a sorted bar). The label IS the insight — put it where the eye lands.

**Tools:** think-cell (if licensed); native Waterfall/Combo charts; data labels, leader lines, callout shapes
