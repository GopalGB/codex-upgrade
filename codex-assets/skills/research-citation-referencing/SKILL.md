# research-citation-referencing

Match the style to the field: **APA 7** (social sciences, author-date), **MLA 9** (humanities, author-page), **Chicago** (notes-bibliography for history/arts, or author-date for sciences), **IEEE** (engineering, bracketed numbers), **Vancouver** (biomedicine, numbered by citation order). The two axes that matter: in-text format (author-date vs numeric) and reference-list order (alphabetical vs citation order).

Don't format by hand. Use **Zotero** (or Mendeley) as the source of truth: capture items via the browser connector, fix metadata at import (capitalization, author parsing), attach DOIs, then let the word-processor plugin or a CSL style render any format on demand. For LaTeX, use BibTeX/biblatex with **Better BibTeX** for stable citation keys.

Always include a **DOI** (as an https://doi.org/ URL) for anything that has one — it's the persistent locator. APA 7 specifics worth knowing: up to 20 authors listed in the reference; "et al." in-text for 3+ authors from the first cite; sentence case for article titles.

Pitfall: trusting Google Scholar's auto-generated citations — they routinely mangle author names, page ranges, and venue; verify against the source. Second pitfall: a citation in text missing from the reference list (or vice versa) — run the manager's consistency check.

**Tools:** Zotero, Mendeley, BibTeX/biblatex, CSL styles, APA 7, IEEE, Chicago author-date, DOI, Better BibTeX
