# prod-focus-distraction-control

Distraction is mostly environmental and internal-trigger driven, not a character flaw. Attack it in layers. **Environment**: phone in another room (proximity alone degrades focus even when silent), single monitor for deep tasks, close all non-essential tabs/apps. **Notification hygiene**: turn off ALL push notifications by default, batch-check messaging in 2-3 scheduled windows, set status to Do Not Disturb during deep blocks. **Internal triggers** (Eyal's *Indistractable*): most distraction is escape from discomfort (boredom, anxiety); when an urge hits, note the emotion, surf the urge for ~10 min, and use a **distraction-capture pad**-write the 'I should check X' thought down to close the loop without acting. Use website/app blockers (Freedom, Cold Turkey) on a schedule so the decision is pre-committed and not relitigated in the moment. Define **traction** (action toward your intent) vs **distraction** (action away) and timebox values into the calendar so you have something concrete to return to. The failure mode is fighting each urge with willpower in real time; pre-commitment and friction beat in-the-moment self-control every time.

**Tools:** notification hygiene, distraction pad, website blockers, traction vs distraction, timeboxing triggers
