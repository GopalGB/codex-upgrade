# prod-note-taking-systems

Two complementary systems. **PARA** (Forte) organizes ALL digital information by actionability into four buckets: **Projects** (active, with a deadline/outcome), **Areas** (ongoing responsibilities-health, finances), **Resources** (topics of interest), **Archive** (inactive). Files move between buckets as actionability changes; organize by where you'll *use* it, not by category. Forte's workflow is **CODE**: Capture (save what resonates), Organize (PARA), Distill (progressive summarization-bold the key lines, highlight the best of the bold), Express (turn notes into output). **Zettelkasten** (Luhmann) is for thinking, not storage: write **atomic** notes (one idea each), in your own words (an 'evergreen' note you'd reuse), each with a unique ID, then **link** them bidirectionally so a web of connections emerges-knowledge compounds via the links, not the folders. Use PARA for project/resource management, Zettelkasten for developing original ideas. The universal failure mode is **collector's fallacy**: hoarding clipped articles you never process-saving isn't learning. Distill and link, or the notes are write-only. Tools (Obsidian, Notion) are interchangeable; the workflow is what matters.

**Tools:** PARA, Zettelkasten, atomic notes, CODE, evergreen notes, progressive summarization, bidirectional links
