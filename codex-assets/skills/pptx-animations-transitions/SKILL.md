# pptx-animations-transitions

Animation builds focus, not flash. Use Animations > Add Animation (not the gallery — it replaces existing). Open the Animation Pane to sequence: set Start to On Click (reveal step-by-step), With Previous (simultaneous), or After Previous (auto-chain). Keep to a tiny vocabulary: Appear/Fade for entrance, a subtle Grow/Color for emphasis, nothing bouncing. Effect Options control direction and, for text, 'By Paragraph' to reveal bullets one level at a time. Triggers (Animations > Trigger > On Click of) fire an animation when a specific object is clicked — good for interactive reveals. Set durations ~0.3-0.5s; longer feels sluggish live. Transitions: pick ONE for the whole deck (Transitions > Apply To All) — Fade or Push; reserve Morph for design moves (separate skill). Pitfall: 'Fly In' + 'Spin' on every bullet screams amateur and slows the speaker; the Animation Pane numbers show how cluttered a slide is — if it reads past ~5, cut. python-pptx does NOT support animations; author them in the app.

**Tools:** Animations tab > Animation Pane; Effect Options; Transitions tab; Trigger; Start On Click/With/After
