# career-linkedin-optimization

LinkedIn Recruiter ranks profiles by keyword relevance + activity + connection distance. **The headline and About section are the highest-weighted search fields** — stuff them with target-role keywords naturally: `Senior Backend Engineer | Go, Kubernetes, Distributed Systems | ex-Stripe`. Don't waste the headline on `Aspiring Developer`.

Levers: (1) custom URL; (2) Skills section — pin the top 3 that match target roles, get endorsements (recruiters filter on skills); (3) About — first 3 lines (the 'see more' fold) must hook, written first-person with a metric-led narrative; (4) Experience bullets mirroring resume impact lines; (5) 'Open to Work' with recruiter-only visibility (the green ring can signal to some hiring managers, so use the discreet recruiter setting); (6) location set to target market.

**What top candidates do differently:** they post/comment 2–3×/week in their niche — activity recency boosts search rank and creates inbound — and they collect a few specific recommendations (recommendations > endorsements for credibility). A custom banner + professional photo lifts profile-view→connection conversion.

**Common mistake:** treating LinkedIn as a static resume mirror. Empty About, default headline, zero activity, generic 'I'd like to add you to my network' connects. The profile is a living, searchable, ranked document — keyword + activity decay both matter.

**Tools:** keyword-rich headline, recruiter boolean search, Open To Work, SSI, About-section story
