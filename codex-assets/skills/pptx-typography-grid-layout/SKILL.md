# pptx-typography-grid-layout

Professional slides come from a grid and a type scale, not vibes. Turn on View > Guides + Gridlines and drag guides to define margins and a column grid; Smart Guides (on by default) snap shapes to edges/centers of neighbors. Use Arrange > Align (Left/Center/Distribute Horizontally) and Distribute to get pixel-true alignment — eyeballing always drifts. Define a type scale: one title size, one subtitle, one body, one caption (e.g. 40/24/18/12) and reuse them — don't free-size text per slide. Left-align body text (centered paragraphs are the #1 amateur tell and the AI-slop signature); reserve centering for short hero lines. Control internal padding via Format Shape > Text Options > Text Box margins and 'Shrink text on overflow' vs 'Do not autofit' (autofit makes fonts inconsistent across slides — prefer fixed sizes and less text). Set line spacing ~1.0-1.15 for headlines, ~1.2 for body. Establish hierarchy with size + weight + color, not 5 fonts. Pitfall: mixing 4 alignments and 6 sizes on one slide. Build alignment into layouts (placeholders are pre-positioned) so every slide inherits the grid for free.

**Tools:** View > Guides/Gridlines; Arrange > Align/Distribute; Format Shape > Text Box margins; Smart Guides; type hierarchy
