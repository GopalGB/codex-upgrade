# prod-deep-work

Deep work (Newport) = professional activity in a state of distraction-free concentration that pushes cognitive limits and is hard to replicate; it's the skill that compounds. Pick a **philosophy** matching your schedule: Monastic (radically isolate), Bimodal (split days/weeks into deep vs open), Rhythmic (a fixed daily block-best for most), Journalistic (deep on demand-only for experts). Build a **depth ritual**: fixed location, fixed duration (start 60-90 min, scale toward ~4 hrs/day max-the brain's ceiling), and a metric (deep-hours tracked on a visible tally). Eliminate the attention-residue tax by going all-in: a 'quick check' of Slack fragments focus for 15+ min afterward. End with a **shutdown ritual** (review tasks, set tomorrow's plan, say a completion phrase) so the brain releases open loops and recovers. The dominant failure mode is treating deep work as willpower rather than scheduled infrastructure-motivation fails; the calendar block and a phone-in-another-room environment succeed. Pair with social-media/notification minimalism: the bottleneck is rarely time, it's protected attention.

**Tools:** deep work, attention residue, depth ritual, shutdown ritual, monastic/bimodal/rhythmic/journalistic
