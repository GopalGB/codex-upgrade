# retail-gmroi-margin

**GMROI = gross margin $ / average inventory cost** — gross-margin return on inventory investment, the buyer's single best profitability metric because it fuses margin AND velocity. GMROI of 3.0 means $3 gross margin per $1 invested in inventory. Equivalent decomposition: **GMROI = gross margin % x inventory turn** (where turn uses cost), so a thin-margin fast-turner can out-earn a fat-margin slow-mover.

Benchmark: ~3.2+ is generally healthy in many retail segments; grocery runs lower-margin/high-turn, jewelry the reverse — judge GMROI vs category peers, not absolute.

The expert move: rank every SKU/category by GMROI to find **cash traps** (good margin %, terrible turn, GMROI <1.5 — they tie up OTB) and protect **cash engines** (modest margin, blistering turn). Distinguish **initial vs maintained margin** — GMROI must use *maintained* (post-markdown, post-shrink) margin or it lies.

Pitfalls: using *retail* inventory value in the denominator instead of cost (overstates inventory, understates GMROI); using initial markup instead of maintained; and optimizing margin % alone, which rewards slow expensive stock. Pair GMROI with turn and sell-through for the full picture; use it to reallocate OTB toward high-GMROI categories.

**Tools:** GMROI formula, gross margin return on inventory, turn x margin, maintained margin
