# prod-smart-goals

SMART turns 'get fit' into an executable target. Each criterion has a test: **Specific**-names the exact what/who/where (not 'improve sales' but 'close 10 enterprise deals in the SaaS vertical'). **Measurable**-has a number and a finish line you can verify objectively. **Achievable**-stretches but is reachable with current/obtainable resources (calibrate against your base rate). **Relevant**-ladders up to a higher objective; if you can't name what it serves, drop it. **Time-bound**-a hard deadline that creates urgency and a review point. The technique that makes SMART actually work: pair the lagging outcome metric ('revenue +20%') with **leading indicators** you control daily ('20 outreach calls/day')-you can't directly move a lagging metric, only the leading ones, so track those. Decompose any goal beyond ~2 weeks into milestones. The dominant failure mode is making goals only Specific/Measurable while skipping Achievable/Relevant-producing precise but pointless or demoralizingly impossible targets. Re-state every goal as one sentence containing all five elements; if you can't, it isn't SMART yet.

**Tools:** SMART criteria, leading vs lagging indicators, goal decomposition
