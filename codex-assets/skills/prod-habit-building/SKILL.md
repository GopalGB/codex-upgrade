# prod-habit-building

Habits run on a neurological loop: **cue -> craving -> routine -> reward** (Duhigg). To install one, design all four. Clear's four laws operationalize it: make it **obvious** (use an implementation intention: 'I will [behavior] at [time] in [location]', or **habit stacking**: 'after [existing habit] I will [new habit]'); **attractive** (pair with something you enjoy-temptation bundling); **easy** (shrink to a 2-minute version and reduce friction-lay out gym clothes the night before); **satisfying** (add an immediate reward and track with a visible streak). To break a bad habit, invert: make the cue invisible, the act hard, the reward unsatisfying. Two operating rules: the **2-minute rule** (start absurdly small so showing up is trivial-mastering the art of showing up beats intensity) and the **never-miss-twice rule** (one miss is an accident, two is the start of a new habit). The core failure mode is relying on motivation and going too big on day one; identity-based framing ('I'm a runner') plus tiny consistent reps beats willpower-driven bursts that collapse.

**Tools:** habit loop, cue-routine-reward, atomic habits 4 laws, habit stacking, implementation intentions
